## 🧠 À propos de ce Portfolio
Ce dépôt contient le code source de mon portfolio professionnel interactif. Conçu pour refléter mon expertise en tant qu'**Ingénieur Logiciel Full-Stack & DevOps / IA**, ce site met en avant une architecture orientée production.

## ✨ Fonctionnalités clés
- **Design Glassmorphism Premium :** Une interface utilisateur moderne, réactive et percutante.
- **Chatbot IA Intégré (RAG) :** Un assistant virtuel interactif intégré directement sur le site.
- **Micro-Animations :** Effets de survol dynamiques et animations fluides (keyframes CSS).
- **Entièrement Responsive :** Adapté pour le mobile, la tablette et le desktop.

## 🛠️ Projets mis en avant
Ce portfolio présente 9 projets concrets majeurs, incluant :
1. **AI RAG Assistant** (LangGraph, Ollama, DevOps)
2. **PMT-AGENT-IA** (Système multi-agents)
3. **AgriSen** (Vision par ordinateur & IA Agricole)
4. Architectures Backend robustes (Laravel, Spring Boot, FastAPI)
